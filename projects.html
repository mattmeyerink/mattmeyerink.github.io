Server Side Instagram Clone

I created a server side Instagram clone that provides all of the basic funcitions of instagram including user authenticaiton as well as the ability to upload posts, follow users, like posts, and comment on posts.

Client Side Instagram Clone

I created a client side Instagram clone that is an extension of the server side instagram clone. The client side version provides client side rendering of the homepage in react using a REST API. This allowed for likes and comments to appear without refreshing the screen as well as double click to like and infinite scroll.


Python | REST API | React | Flask | SQL | HTML | CSS | Git | Bash



Dinner Schedule Generator

This is a web application that generates a unique weekly dinner schedule for the user. The dinner schedule incudes the name, ingredients, and the source for the meal. After the schedule is generated, the user can edit any meal to be a custom meal of their choosing. After the user is satisfied with the meals they can submit the meals to a pdf that is automatically downloaded.


Javascript | React | API | HTML | CSS | Git



Harry Potter Adventure Game

This is an adventure game that uses Tkinter to create a GUI to emerse the user into an accurate Harry Potter experience where the user must direct their character to pick up items to kill all of the Horcruxes before attempting to kill Lord Voldemort and save the Wizarding World.


Python | Tkinter | Git
