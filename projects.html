Instagram Clone

I created two versions of this project. First I created a server side version that interacts with a SQL database to execute all of the basic functions of instagram including user authentication as well as the ability to upload posts, follow users, like posts, and comment on posts. It also allows users to delete only their own posts and comments and edit their account information through their profile page.

Next I created a client side version that is an extension of the server side Instagram clone. The client side version provides client side rendering of the homepage in React referencing a REST API I created. This allowed for likes and comments to appear without refreshing the screen as the addition of double click to like and infinite scroll functionalities.

Python | REST API | React | Flask | SQL | HTML | CSS | Git | Bash

Dinner Schedule Generator

This is a web application that generates a unique weekly dinner schedule for the user. The dinner schedule incudes the name, ingredients, and the source for the meal. After the schedule is generated, the user can edit any meal to be a custom meal of their choosing. After the user is satisfied with the meals they can submit the meals to a PDF that is automatically downloaded.

Javascript | React | Web API | HTML | CSS | Git
